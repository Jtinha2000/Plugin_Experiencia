# Plugin Experiencia

> Esse repositório contém uma extensão desenvolvida para o jogo Unturned. Nela há a implementação de uma barra de XP à parte, responsável por liberar recompensas pré-configuradas.

## ⚠️ Aviso

O projeto foi desenvolvido unicamente com fins recreativos, não há previsão para atualizações, futuros reparos nem qualquer garantia de qualidade ou segurança! Utilize a aplicação por sua conta e risco.
